timer (func, interval, run_times)
Creates a new timer object and starts it The timer runs the function func every interval seconds. It does this up to run_times. After the timer is created, the function is run after the given interval, not immediately.

Parameters

  • func: The lua function to execute
  • interval: The interval, in seconds
  • run_times: The maximum number of times to execute func, 0 for no limit

Usage:

timer(function () print("hi") end, 1)

Return value:

the new timer object
