background (file, style)
Set the background to an image or solid color

Parameters

  • file: Either the filename of an image or the string "color" for a solid color background
  • style: How to display the image; either "stretch", "zoom", or "center". If file was "color", then this parameter should be a color name or hex code.

Usage:

background("foo.png", "stretch") or background("color", "#bababa")
