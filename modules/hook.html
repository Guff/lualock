connect (hook_name, func)
Connects a lua function to a given hook

Parameters

  • hook_name: The hook to connect to; one of "lock", "unlock", or "auth-failed"
  • func: The function to run when the hook is triggered

Usage:

hook.connect("lock", function () print("THE SCREEN IS LOCKED!") end)
