clear (self)
Clears the image buffer

Parameters

  • self: The image to clear

foo:clear()
draw_circle (self, x, y, radius, fill, color)
Draws a circle on the image

Parameters

  • self: The image
  • x: x coordinate of the circle's center
  • y: y coordinate of the circle's center
  • radius: radius of the circle
  • fill: true to fill the circle with color, false to draw the outline
  • color: color to draw the circle with

Usage:

foo:draw_circle(0, 0, 50, false, "#ff0000")
draw_line (self, x1, y1, x2, y2, width, color)
Draws a line on the image

Parameters

  • self: The image
  • x1: x coordinate of the line's starting point
  • y1: y coordinate of the line's starting point
  • x2: x coordinate of the line's ending point
  • y2: y coordinate of the line's ending point
  • width: width of the line
  • color: color of the line

Usage:

foo:draw_line(0, 0, 100, 100, 5, "#000000")
draw_rectangle (self, x, y, width, height, fill, color)
Draws a rectangle on the image

Parameters

  • self: The image
  • x: x coordinate of the rectangle's top left corner
  • y: y coordinate of the rectangle's top left corner
  • width: width of the rectangle
  • height: height of the rectangle
  • fill: true to fill the rectangle with color, false to draw the outline
  • color: color to draw the rectangle with

Usage:

foo:draw_rectangle(0, 0, 100, 100, true, "#ff0000")

rotate (self, angle, x, y)
Rotates the image

Parameters

  • self: The image to rotate
  • angle: Angle, in degrees, to rotate the image by
  • x: x coordinate to use as the image's center of rotation. Default: 0
  • y: y coordinate to use at the image's center of rotation. Default: 0

Usage:

foo:rotate(60)
scale (self, sx, sy)
Scales the image

Parameters

  • self: The image to scale
  • sx: The ratio to scale the image's width by
  • sy: The ratio to scale the image's height by

Usage:

foo:scale(0.5, 0.5)
