Module text

Functions

draw (self) Draws the text object
hide (self) Hides the text object
set (self, args) Sets the text object's properties Note that for the changes to be reflected on screen, you must call draw() after this.

text (args)
Creates a new text object Note that draw() must be called to display the text

Parameters

  • args: Table of properties to use for the text object; see properties table

Usage:

bar = text({ text = "User: " .. os.getenv("USER"), x = 500, y = 370 })

Return value:

The new text object

Tables

properties
Available text object properties Fields
  • text: The text to display
  • x: x position of the text
  • y: y position of the text
  • font: font to use; of the form "Sans Bold 12"
  • color: color of the text
  • border_color: color of the text's border
  • border_width: width of the text's border
