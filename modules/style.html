style (color, font, x, y, off_x, off_y, width, height)
Set style of the password field

Parameters

  • color: Color of the password text. Should either be a hex string of the form "#ff0000" or a color name like "red". Default: "#000000"
  • font: Font to use for the password text. Should be of the form "Sans Bold 12". Default: "Sans 12"
  • x: x position of the field's top left corner relative to the top left corner of the screen.
  • y: y position of the field's top left corner relative to the top left corner of the screen.
  • off_x: Horizontal offset of the text within the password field.
  • off_y: Vertical offset of the text within the password field.
  • width: Width of the password field in pixels.
  • height: Height of the password field in pixels.

Usage:

style({ color = "#baddab", font = "Droid Sans 10", x = 100, y = 200 })
